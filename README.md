# RollingCat Software — Portfolio Website

The official portfolio website for **RollingCat Software**, showcasing modern web, mobile, and AI solutions.

**Live:** [rollingcatsoftware.com](https://rollingcatsoftware.com)

## About

A single-page portfolio website for Ahmet Abdullah Gultekin — a full-stack developer and Computer Engineering student based in Istanbul. The site showcases 6 projects, 12+ technologies, and a self-hosted infrastructure stack.

## Tech Stack

- **HTML5** — Semantic markup with bilingual support (EN/TR)
- **CSS3** — Custom dark theme, CSS Grid, responsive design, animations
- **Vanilla JavaScript** — IntersectionObserver animations, language toggle, mobile menu

No frameworks, no build step — just fast, lightweight HTML.
